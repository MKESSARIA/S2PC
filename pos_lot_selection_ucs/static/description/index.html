Overview

  • This module helps you to check quantity or reduce quantity by lot/serial directly from Point of Sale interface.

Features

Using this module you can select product by checking lot/serial and directly you can reduce quantity in point of sale screen.

Using this module you can check product lot/Serial Expirydate and if product is expiry that system will not allow you to reduce qunaity of that lot/serial product in point of sale screen.

Addional Option to Print Lot/Serial Number On POS Receipt.

Compatible with both Community and Enterprise version of odoo.
